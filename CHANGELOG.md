# Changelog

## [1.0.0] — 2026-06-20

### Added
- Streamlit UI with chat management
- Document upload (PDF, DOCX, TXT, CSV, MD, images)
- OCR support for scanned images via Tesseract
- Find-Retrieve-Answer workflow with Ollama (Qwen2.5:1.5b)
- SQLite persistence for chats, sources, and messages
- Section-based document splitting by headings
- Pre-commit hooks (ruff, pyrefly, pytest)
- AGPLv3 license
- CI pipeline configuration
