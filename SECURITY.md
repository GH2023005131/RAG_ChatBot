# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening an issue on the repository.

Do not disclose the vulnerability publicly until it has been addressed.

## Local-Only Design

DocVision AI is designed to run entirely locally. No data is sent to external servers. The application does not collect telemetry or analytics.
